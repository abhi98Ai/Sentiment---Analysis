# Sentiment---Analysis
Sentiment Analysis of Amazon product reviews with data preprocessing ,visualization and business insights .

## Project Overview
This project performs Sentiment Analysis on Amazon product reviews using Python and TextBlob. It cleans review text, classifies reviews into Positive, Negative, and Neutral sentiments, and visualizes the results using bar and pie charts.

## Features
- Text Preprocessing
- Sentiment Analysis using TextBlob
- Bar Chart Visualization
- Pie Chart Visualization
- Sentiment Percentage
- Business Insights
- CSV Export

## Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib
- NLTK

## Project Output
- Sentiment Classification
- Bar Chart
- Pie Chart
- Business Insights
- Exported CSV File
